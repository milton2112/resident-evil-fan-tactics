# Publicar con GitHub Pages

Link esperado:

https://milton2112.github.io/resident-evil-fan-tactics/

Pasos:

1. Abrir `Settings` del repo.
2. Entrar a `Pages`.
3. En `Source`, elegir `Deploy from a branch`.
4. En `Branch`, elegir `main` y carpeta `/ root`.
5. Guardar.
6. Esperar 1 o 2 minutos y abrir el link.

Si el link da 404, esperar un poco mas o refrescar la pagina de Pages hasta que GitHub muestre el estado `Your site is live`.
