# Zona Cero Tactica - Prototipo

Primer prototipo jugable fan inspirado en Resident Evil: tactico por turnos, vista 2.5D isometrica, escuadron humano contra infectados.

## Como probar

Abrir `index.html` en el navegador.

## Controles

- Seleccionar un soldado del tablero o del panel.
- Elegir `Mover` para desplazarlo dentro de su rango.
- Elegir `Atacar` y seleccionar un infectado en rango.
- Usar `Especial` una vez por mision para golpear varios infectados cercanos.
- Cambiar armas desde el panel lateral.
- Usar `Esperar` o `Terminar turno` para pasar accion.
- Elegir `Umbrella` en la pantalla inicial para comprar B.O.W.s con presupuesto antes de iniciar.
- Elegir misiones, ganar recompensas y desbloquear la siguiente zona.
- El progreso queda guardado en el navegador con `localStorage`.
